# Seekr App 🔍

A web-based **Lost and Found System** designed for schools to help students and staff easily report, search, and recover lost items through a centralized platform.

---

## 🚀 Features

* 📢 Report lost items
* 📦 Submit found items
* 🔎 Search and filter items
* ✅ Claim and manage item status
* 👤 User-friendly interface

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Backend:** PHP
* **Database:** MySQL

---

## 📂 Project Structure

```
seekr-app/
├── frontend/     # React application
├── backend/      # PHP API / server

```

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/your-username/seekr-app.git
cd seekr-app
```

### 2. Setup Frontend (React)

```
cd frontend
npm install
npm run dev
```

### 3. Setup Backend (PHP)

* Place backend folder in your local server (e.g., XAMPP htdocs)
* Start Apache & MySQL

### 4. Setup Database

* Import the `.sql` file into MySQL
* Update database credentials in your PHP config file
